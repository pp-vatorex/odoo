WebP Image Optimizer

Speed up your website with WebP Image Optimizer and get lighter images without losing quality.

You can now use this power directly in Odoo.

After enabling it, all your existing images will be automatically optimized also when uploaded into Odoo.

No In App Purchases.

Features

  • Converts all existing images PNG, JPG, GIF Animations into WebP Format.
  • Serve WebP images using HTML 5 Picture Tag
  • Significant Website load time savings.
  • Multi Website ready.

Usage

Install the following python dependencies before you install the module

pip3 install webp
pip3 install cssselect

1. After installation go to Website App -> Configurations -> Websites from menu.

2. Select the website to be converted.

3. Enable WebP for current website.

4. Set Quality for WebP images.
